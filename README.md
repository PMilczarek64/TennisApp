### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/PMilczarek64/TennisApp.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
